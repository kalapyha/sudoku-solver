# REACT Sudoku Solver

> Sudoku Solver app built with REACT.

![](sudoku-app.png)

## Quick Start

Make sure you've installed node

```bash
# Install dependencies
npm install

# Run the React app (client)
npm run start

# Run test
npm run test

# App runs on http://localhost:3000
```
